# Sound Files Directory

This directory contains audio files for lesson completion celebrations.

## Required Sound Files

Add the following MP3 files (each <50KB):

- `completion.mp3` - Main completion sound (uplifting chime)
- `achievement.mp3` - Achievement unlock sound (short fanfare)
- `perfect.mp3` - Perfect score sound (triumphant melody)
- `star.mp3` - Star earned sound (sparkle effect)
- `level_up.mp3` - Level up sound (optional)

## Sources for Royalty-Free Sounds

- [Freesound.org](https://freesound.org)
- [Zapsplat](https://www.zapsplat.com)
- [Mixkit](https://mixkit.co/free-sound-effects/)
- Generate programmatically using Web Audio API

## File Requirements

- Format: MP3
- Size: <50KB each
- Sample rate: 44.1kHz
- Bit rate: 128kbps or lower
